About us

This website started as a fun side-project to become more familliar with the Angular framework, while also keeping an eye out for styling and layout for the main pages at least ;-). Maybe it could be an inspiration source of some kind.


Project info

Created the 29th of may 2024

Angular version 18 (Initial startup)

Command ng new --no-standalone --ssr

This project is server side rendered and components are not standalone by default. They can be made standalone however by setting the "standalone" property to true in the component metadata.
